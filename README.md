# Movie-Recommendation-System
This movie recommendation system is based on content based movie recommendation system


## Collected Data
-Data used for project was downloaded from kagel <a href="https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata" target="_blank">Movies database</a>

## Preprocessing and cleaning of data
-Clean the data and remove any irrelevant information like movie titles, cast, etc. Keep only the relevant information like genre, plot, director, and ratings.

## Feature Extraction
- Use Natural Language Processing (NLP) techniques to extract features from the plot. You can use techniques like Bag of Words or TF-IDF to convert the plot into a numerical representation.

## Similarity Calculation(Cosine similarity)
-Calculate the similarity between movies based on their feature representation. You can use techniques like cosine similarity to calculate similarity.

## Recommender system
- Finally, recommend movies based on the similarity scores. For example, recommend the top 5 movies with the highest similarity score to a given movie.
